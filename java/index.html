jdk 源码阅读

1. java源码阅读环境搭建

  1. 解压 C:\Program Files\Java\jdk1.8.0_131\src.zip , 重命名jdk_source

  2. 打开idea，选择Project -> File -> Project Structure -> SDKs -> Sourcepath,添加并选择刚才解压的目录

3）编辑器左侧会出现java的源码包，选择要看的类的源码就能进去

1.2 IDEA 调试java 源码技巧

  1. 查看接口实现类：Ctrl + Alt +B 选中按快捷键，然后跳到实现类地方去
  2. 查看当前类继承树：Ctrl + H

2. 源码部分

2.1 Object 类
  1. hashCode
    • public native int hashCode() Object 类下的native 方法，返回对象哈希值
    • equals 比较两个对象相等，则一定有相同哈希值，equals比较二者不相等，则哈希码也可能相等，但是不等对象产生不同整型结果有助于提高哈希表性能
  2. clone
    • protected native Object clone() throws CloneNotSupportedException 创建并返回对象的副本
    • 该方法是“浅拷贝”，该方法将创建该对象类的一个新实例，并使用该对象相应字段的内容初始化所有字段，就像赋值一样，这些字段的内容本身不是克隆的
    • Object类本身没实现Cloneable 接口，如果一个类没有实现Cloneable接口，就会抛出CloneNotSupportedException异常，
2.2 Serializable 接口
  • 是一个接口，仅标识可序列化的语义
